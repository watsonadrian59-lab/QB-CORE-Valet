# QB-CORE-Valet
This is a valet script that i made for my server after not seeing many free ones being released. This script is for QBCORE Servers ONLY!

Dependencies:

- QB-MENU
- QB-INPUT
- QB-CORE
- OXLIB


Very lightweight, standalone qb-menu valet script.

/valet

OR

F6 keybind.

STEPS:

Download file.

Extract "qb-valet".

Drag and drop into [standalone] OR Folder of your choosing.

Refresh and ensurethe resource.

ENJOY!
